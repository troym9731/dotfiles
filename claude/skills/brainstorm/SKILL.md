# Brainstorm

Help the user brainstorm different solutions before implementing a plan. Ask
clarifying questions when constraints are too vague or unclear. Utilize the
`search` skill to discover necessary information across directories/codebases or
the internet in order to acquire relevant information for the brainstorming session.

## Output

Once the user is satisfied with the brainstorming session, output the results
to `~/instinct/claude_brainstorms` using the format in [template.md](references/template.md).

Name the file `YYYY-MM-DD-brainstorm-<topic>.md` where `<topic>` is a short kebab-case summary.
