# Plan

Help the user build an actionable plan. Ask clarifying questions to narrow scope
and surface constraints before committing to steps. Use the `search` skill when
context from codebases or the internet is needed.

## Output

Once the plan is finalized, output the results to `~/instinct/claude_plans`
using the format in [template.md](references/template.md).

Name the file `YYYY-MM-DD-plan-<topic>.md` where `<topic>` is a short kebab-case summary.
